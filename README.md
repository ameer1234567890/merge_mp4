#### Works with
* Almost any linux based shell.
* Cygwin shell on Windows.
* Git Bash on Windows.

#### Requirements
* ffmpeg
* Some patience.

#### Setup Instructions
* Copy `merge_mp4.sh` to a folder containing all files that you need to merge. Note that files need to be in alphabetic order.
* Run `./merge_mp4.sh`.
